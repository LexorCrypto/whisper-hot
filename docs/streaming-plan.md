# Streaming транскрипция — план реализации

Статус: ОТЛОЖЕНО
Приоритет: P3 XL
Дата: 2026-04-16

## Суть

Сейчас WhisperHot записывает весь аудио, потом отправляет batch-запросом.
Streaming позволит показывать текст по мере речи (как в SuperWhisper).

## Что нужно реализовать

### 1. WebSocket клиент (L)
- Новый `StreamingSTTProvider` протокол
- WebSocket соединение к Deepgram/AssemblyAI
- Аутентификация через API-ключ
- Обработка промежуточных и финальных результатов
- Реконнект при обрыве

### 2. Переделка AudioRecorder (L)
- Сейчас: пишет WAV на диск, отдаёт URL после stop
- Нужно: отправлять аудио-чанки по WebSocket во время записи
- Формат: PCM 16kHz mono (как сейчас), но по частям каждые ~100ms
- Сохранить возможность batch-записи (переключатель)

### 3. UI промежуточного текста (M)
- Показывать текст в floating capsule по мере речи
- Промежуточный текст серым, финальный белым
- Расширить capsule по высоте для текста
- Анимация появления слов

### 4. Настройки (S)
- Переключатель streaming/batch в Providers
- Выбор streaming провайдера (Deepgram/AssemblyAI)
- API-ключ для streaming провайдера
- Keychain account для Deepgram/AssemblyAI

### 5. Интеграция (M)
- MenuBarController: выбор между batch и streaming flow
- Пост-обработка после финального результата (как сейчас)
- Context routing работает одинаково
- Word replacements применяются к финальному тексту

## Архитектурные решения

- WebSocket через URLSessionWebSocketTask (встроенный в Foundation)
- Streaming и batch — два разных flow, переключаемых в настройках
- Промежуточные результаты не сохраняются в историю
- Пост-обработка только после финального результата

## Зависимости

- API-ключ Deepgram или AssemblyAI
- Интернет (streaming только облачный, без офлайн)
- Accessibility permission (для показа текста в capsule)

## Когда начинать

Текущий batch-подход (Groq $0.90/мес, задержка 0.5 сек) покрывает
основные use cases. Streaming имеет смысл когда:
- Пользователи жалуются на задержку
- Нужна реал-тайм транскрипция для длинных диктовок
- Появляется конкуренция с другими STT-приложениями
